## Команды для запуска тестов:

```bash
# Переход в папку тестов
cd AirlineTicketSystem.Tests

# Запуск всех тестов
dotnet test

# Запуск с подробным выводом
dotnet test --verbosity normal

# Запуск конкретного теста
dotnet test --filter "WorkingTest"
```
